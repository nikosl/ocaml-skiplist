# skiplist

A SkipList behaves as a sorted list with, typically, O(log(n)) cost for insertion, look-up and removal

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Skiplist.greet "World"
```
